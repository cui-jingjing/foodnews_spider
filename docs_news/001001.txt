 快过年了
 你家年货备齐了吗？
 特别是瓜子、花生、开心果这些坚果
 永远是大家寂寞嘴巴的“最佳伴侣”！
 但是
 接下来这个检测结果
 恐怕你得看仔细了！
 记者网购了5款“网红”坚果
 1号：某品牌焦糖味花生
 2号：一款奶香味巴旦木
 3号：某著名品牌开心果
 4号：某国外品牌黄油腰果
 5号：某款混合型每日坚果
 根据食品安全规范，专家对坚果中5个重要质量指标进行抽检，分别是二氧化硫、亚硝酸盐、甜蜜素、黄曲霉毒素B1和霉菌含量。
 检测结果显示，5份样品，都不含有黄曲霉毒素B1和亚硝酸盐，但都含有微量二氧化硫，其中2号奶香巴旦木的二氧化硫含量最高，但没有超过国家规定的400mg/kg范围。
 令人意外的是，试吃时人气最高的4号蜂蜜黄油腰果，明明外包装配料表上没有标注使用甜蜜素，却是所有样品中，甜蜜素含量最高的一个。不过，5份样品甜蜜素都没有超标。
 \网红\坚果霉菌超标16倍！
 随后，检验员对霉菌含量进行检测，经过48小时观察，惊人的结果出现了。
 1到4号样品试纸上，没有明显霉菌生长现象，而5号每日坚果试纸上，却出现了大量霉菌斑点。
 经计算，每克中含有约400个霉菌菌落。比起国家规定霉菌限量标准：每克不得超过25个霉菌菌落，5号样品的霉菌含量，整整超标了16倍！
 16倍！
 霉菌超标或致癌
 吃多了可能急性中毒
 专家介绍，霉菌超标的坚果，不仅容易腐败变质，部分霉菌会产生毒素，过量食用可能有致癌风险。通常霉菌超标，都与企业储存、运输不当有关。一次吃太多，很有可能急性中毒。
 图文无关
 目前，江苏省食药监部门表示将介入调查。
 提醒大家
 选择正规商家购买年货
 注意查看外包装上的标签、生产标准号
 尽可能选择日期新鲜的
 新鲜坚果、瓜子
 闻起来都会有果实的清香味
 如果闻起来有霉味
 或者不新鲜的油脂味儿
 也称“哈喇味”的
 不要买！！！
 「
 这些干果不能要
 」
 / 漂白核桃 /
 正常核桃经过水洗后颜色偏黑灰，不太讨喜，一些商家为了核桃卖相更好，会使用双氧水甚至是氨水对核桃进行漂白，甚至将前一年的核桃重新加工后再出售。
 选购核桃时，不要买外表太过干净的核桃，核桃仁以白色为佳，如果看到核桃仁上有一层油脂，或者核桃仁已经变黑，千万不能要。
 / 漂白开心果 /
 开心果是很多人过年常备的干果，正常的开心果，果壳偏黄，果仁为绿色，果皮呈紫红色。果壳过分均匀白净的开心果，就不要购买啦！
 / 染色瓜子 /
 市面上的瓜子不仅有各种不同口味，还有五颜六色的外壳，让人眼花缭乱。然而这些不同颜色的瓜子往往添加了大量香精和香料，使用了不同色素，有的染色瓜子搓一搓，就能搓下外壳颜色，吃起来危害不小。
 因此，选购瓜子时选择原味、新鲜的，散发着淡淡的干果香味的瓜子最好。
 / 变质花生 /
 坚果含有大量油脂，不注意存放很容易变质，产生一股“哈喇子味”，黑心商家会用变质花生作为原料加工，并且添加大量奶精和去麻粉，麻痹舌头，让人吃不出变质味道。
 选购花生时可以先尝尝，有“哈喇子味”或者闻起来很香，但吃起来酸涩的坚决不能要。
 / 陈年杏仁 /
 有些杏仁颜色发暗偏深，颗粒干瘪个头偏小，极有可能是陈年的杏仁，或者使用化学物品加工过，很多营养都流失了，没有食用价值。注意啦，一些果仁上有小洞的是虫蛀粒，有白色斑点的杏仁已经发霉了，千万不能食用！
 新鲜杏仁颜色较浅，个头又大又饱满，吃起来清新爽口，没有“哈喇子味”，这才是值得大家选购的！
 / 滑面板栗 /
 一些表面看起来特别光滑并且有光泽度的板栗，大家选购时要留个心眼，它们有可能用石蜡加工过。有的商家为了增加板栗甜度，加工时还会添加甜蜜素，大家选购时，一定要注意挑选那些光泽感、甜度都正常的板栗。
 大家一定要小心辨别，仔细斟酌！
